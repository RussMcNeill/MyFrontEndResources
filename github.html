Working with GitHub Repo (Repository)

*Step 1* - create a repo in Github and clone it down into your local machine - `git clone yourgithuburl.com`

*Step 2* - add or create your files in the cloned folder

*Step 3* - add and commit your code locally - `git add .` then `git commit -m "some message about what you did"`

*Step 4* - push your code to Github - `git push origin master`
